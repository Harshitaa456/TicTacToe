# 🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Live Demo

👉 [Play Here](https://tictactoe456.netlify.app/)

## 📁 Features

- Classic 3x3 Tic Tac Toe
- New Game button to reset
- Shows winner or draw
- Responsive design

## 🛠️ Tech Used

- HTML
- CSS
- JavaScript
